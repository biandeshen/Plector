# 问题：Python 找不到 `core` 模块

需要在 `websocket.py` 顶部添加项目根目录到 `sys.path`。

### 修复

在 `channels/websocket.py` 最顶部（import 之前）添加：

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
Plector WebSocket 渠道 + Dashboard
"""

import sys
from pathlib import Path

# 将项目根目录加入 sys.path
sys.path.insert(0, str(Path(__file__).parent.parent))
```

### 验证

```bash
python channels/websocket.py --port 8080
```

### 提交

```bash
git add channels/websocket.py
git commit -m "fix(channels): websocket.py 添加 sys.path 修复模块导入"
```
