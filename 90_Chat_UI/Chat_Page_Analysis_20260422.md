## 4. 所有按钮功能

### Header 区域

| 按钮 | 位置 | 功能 |
|------|------|------|
| Plector Logo | Header左侧 | 返回首页 |
| 监控 | Header导航 | 跳转 Dashboard (/) |
| 对话 | Header导航 | 跳转 Chat (/chat) |
| ☀️ | Header右侧 | 切换深/浅主题 |

### 侧边栏

| 按钮 | 功能 |
|------|------|
| 新对话 | 清空当前对话，创建新会话 |
| 搜索对话... | 过滤历史对话列表 |
| ✕ (每个历史项) | 删除该对话 |
| 历史对话项 | 点击加载该对话的消息 |

### 快速入口（6个）

| 按钮 | 功能 |
|------|------|
| 📝 帮我写一个 Python 脚本 | 自动填入提示词并发送 |
| 🔍 解释这段代码做了什么 | 自动填入提示词并发送 |
| 🌐 搜索最新的 AI 新闻 | 自动填入提示词并发送 |
| 🏗️ 分析这个项目的架构 | 自动填入提示词并发送 |
| 💚 帮我检查系统健康状态 | 自动填入提示词并发送 |
| ▶️ 继续之前的任务 | 继续上次的任务 |

### 主聊天区域

| 按钮 | 功能 |
|------|------|
| 发送 | 发送输入框中的消息 |
| 重新生成 | (AI回复下方) 重新执行推理 |
| 复制 | (AI回复下方) 复制回复内容 |

## 6. 问题发现

### 已修复

| 问题 | 状态 | 说明 |
|------|------|------|
| NaN 时间 | ✅ 已修复 | 使用 `MAX(timestamp)` 替代 `created_at` |
| 流式中断 | ✅ 已修复 | buffer 阈值从 30 降到 10 |
| 消息不保存 | ✅ 已修复 | 后端正确保存消息 |

### 待修复

| 问题 | 状态 | 根因 |
|------|------|------|
| 思考内容为空 | ⚠️ 待验证 | MiniMax thinking 格式可能不匹配 |
| 工具调用对话丢失 | ⚠️ 待验证 | 刷新后 tool_call 消息未加载 |
