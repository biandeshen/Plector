#### 新建模块

```python
# core/path_manager.py
class PathManager:
    PROJECT_ROOT = Path(__file__).parent.parent.resolve()

    @classmethod
    def data_dir(cls) -> Path:
        return cls.PROJECT_ROOT / "data"

    @classmethod
    def db_path(cls) -> Path:
        return cls.data_dir() / "plector.db"
```

### 3.6 worktree/memory-system (P2)

**优先级**: 🟡 中
**依赖**: architecture-foundations + llm-optimization
**目标**: 实现艾宾浩斯遗忘曲线 + 联想式记忆

#### 艾宾浩斯遗忘曲线

```
阶段一：基础实现
├── 添加记忆强度等级 (alive/normal/fading/forgotten)
├── 实现衰减系数 (5/10/30/50 轮次)
└── 对话轮次触发衰减检查

阶段二：触发机制
├── 时序衰减 (时间维度)
├── 频次衰减 (访问次数)
└── 重要性衰减 (手动标记)
```

#### 八种联想模式

| 模式 | 说明 | 优先级 |
|------|------|--------|
| 语义相似 | 向量余弦相似度匹配 | P1 |
| 触景生情 | 上下文关键词触发 | P2 |
| 触类旁通 | 关联知识扩展 | P2 |
| 时序联想 | 时间维度关联 | P2 |
| 温故知新 | 间隔重复提醒 | P2 |
| 举一反三 | 场景生成 | P3 |
| 推陈出新 | 知识综合 | P3 |
| 恍然大悟 | 冲突检测 | P3 |
