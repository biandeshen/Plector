# 问题：agent 未初始化

`agent` 是全局变量，初始值为 `None`。`uvicorn.run()` 启动后没有调用初始化逻辑。

### 修复

在 `channels/websocket.py` 中添加 FastAPI 启动事件：

找到 `# 创建 FastAPI 应用` 那一行，在下面添加：

```python
# 创建 FastAPI 应用
app = FastAPI(title="Plector", version="1.1.0")


@app.on_event("startup")
async def startup():
    """应用启动时初始化 Agent"""
    global agent
    if agent is None:
        agent = AgentLoop()
        logger.info("Agent 已初始化")
```

然后修改 `main()` 函数，删除 `agent = AgentLoop()` 那一行，改为：

```python
def main():
    """启动服务器"""
    parser = argparse.ArgumentParser(description="Plector WebSocket 渠道")
    parser.add_argument("--host", default="127.0.0.1", help="监听地址")
    parser.add_argument("--port", type=int, default=8080, help="监听端口")
    args = parser.parse_args()

    logger.info(f"Plector WebSocket 启动: http://{args.host}:{args.port}")
    logger.info(f"Dashboard: http://{args.host}:{args.port}/")
    logger.info(f"WebSocket: ws://{args.host}:{args.port}/ws")

    uvicorn.run(
        "channels.websocket:app",
        host=args.host,
        port=args.port,
    )
```
