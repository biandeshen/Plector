## 四、未来升级方案（优先级排序）

### P0 - 紧急（核心差距）

#### 1. 多用户隔离 (multi-tenant)

```
┌─────────────────────────────────────────────────────────┐
│ 问题：当前只能单机自用，无法服务多人                      │
│                                                         │
│ 工作量：较大                                            │
│                                                         │
│ 影响：决定能否商业化部署                                 │
│                                                         │
│ 关键文件：                                              │
│   - channels/websocket.py                              │
│   - core/agent_loop.py                                 │
│   - skills/memory/implementation.py                     │
└─────────────────────────────────────────────────────────┘
```

#### 2. 联想式记忆 + 艾宾浩斯遗忘曲线

```
┌─────────────────────────────────────────────────────────┐
│ 问题：当前记忆系统缺少衰减机制，8种联想模式未实现         │
│                                                         │
│ 关键文件：                                              │
│   - core/vector_memory.py（需添加艾宾浩斯曲线）         │
│   - skills/memory/implementation.py（需添加联想模式）   │
│                                                         │
│ 前置条件：先修复补丁文档55中的缺陷                       │
└─────────────────────────────────────────────────────────┘
```

### P1 - 重要（体验提升）

#### 3. 记忆管理面板

| 功能 | 说明 |
|------|------|
| 查看记忆 | 用户可查看所有记忆条目 |
| 删除记忆 | 选择性删除不需要的记忆 |
| 编辑记忆 | 手动修正记忆内容 |
| 开关控制 | 用户可开启/关闭记忆功能 |

#### 4. 对话分支

| 功能 | 说明 |
|------|------|
| 历史节点 | 支持回到任意对话节点 |
| 重新回答 | 基于历史上下文重新生成回复 |
| 分支管理 | 可命名、切换、删除对话分支 |

#### 5. 消息操作

| 功能 | 说明 |
|------|------|
| 复制 | 一键复制消息内容 |
| 编辑 | 可修改已发送消息并重新生成 |
| 重新生成 | 基于相同上下文重新生成回复 |
| 引用标注 | 在回答中标注信息来源 |

#### 6. 用户画像

| 功能 | 说明 |
|------|------|
| 偏好学习 | 根据历史对话自动推断用户偏好 |
| 画像展示 | 可视化展示用户特征 |
| 画像编辑 | 手动修正或补充用户画像 |

## 五、实施路线图

### 阶段一：基础完善（1-2周）

```
┌─────────────────────────────────────────────────────────┐
│ 1. 修复补丁文档55中的所有缺陷                            │
│    - import uuid 缺失                                   │
│    - 旧记忆兼容处理                                      │
│    - 补全 save_conversation 等方法                      │
│                                                         │
│ 2. 验证现有功能正常工作                                  │
│    - 向量搜索                                            │
│    - 记忆存储                                            │
│    - 上下文保鲜                                          │
└─────────────────────────────────────────────────────────┘
```

### 阶段二：记忆系统增强（2-3周）

```
┌─────────────────────────────────────────────────────────┐
│ 1. 实现艾宾浩斯遗忘曲线                                 │
│    - 基于对话轮次的衰减机制                              │
│    - 稳定性系数（5/10/30/50）                          │
│    - 记忆强度等级（alive/normal/fading/forgotten）      │
│                                                         │
│ 2. 实现8种联想式记忆模式                                │
│    - 语义相似                                            │
│    - 触景生情（上下文触发）                              │
│    - 触类旁通（关联扩展）                                │
│    - 时序联想                                            │
│    - 温故知新（复习提醒）                                │
│    - 举一反三（场景生成）                                │
│    - 推陈出新（知识综合）                                │
│    - 恍然大悟（冲突检测）                                │
│                                                         │
│ 3. 添加记忆管理面板                                      │
└─────────────────────────────────────────────────────────┘
```

### 阶段三：体验优化（持续）

```
┌─────────────────────────────────────────────────────────┐
│ 1. 对话分支功能                                         │
│ 2. 消息操作增强（复制/编辑/重新生成）                   │
│ 3. 界面改造（Dashboard/Chat）                           │
│ 4. 多用户隔离（商业化准备）                             │
└─────────────────────────────────────────────────────────┘
```
