## 六、关键决策

### 6.1 技术选型

| 决策点 | 方案 | 理由 |
|--------|------|------|
| 中间件框架 | 自研 MiddlewareChain | 轻量可控，与 AgentLoop 深度集成 |
| FTS5 实现 | SQLite FTS5 | 零依赖，与现有 DB 统一 |
| LLM 提取 | 调用现有 LLM | 复用现有资源 |
| 工作流执行 | LangGraph StateGraph | 成熟稳定，持久化支持 |
| 事件总线 | 增强 EventBus V2 | 已有基础，弱引用防泄漏 |

### 6.2 向后兼容

- 保留所有现有 skill.json 接口
- 保留所有现有 MCP 工具
- 新增接口通过 Bridge 桥接
- 联动链可选启用
