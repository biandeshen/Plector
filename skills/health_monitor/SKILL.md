# Health Monitor Skill

## 目的
检查系统资源使用情况，返回 CPU、内存、磁盘使用率和健康状态。

## 适用场景
- 用户询问系统是否正常
- 用户询问资源使用情况
- 定期健康检查

## 执行步骤
1. 调用 check_health 工具
2. 返回 CPU、内存、磁盘使用率
3. 判断状态（healthy / degraded）

## 成功标准
- 返回 CPU 使用率
- 返回内存使用率
- 返回磁盘使用率
- 返回健康状态

## 相关工具
- `check_health`：执行健康检查，返回 CPU、内存、磁盘使用率和健康状态
