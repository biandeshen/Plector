# Web Search Skill

## 目的
搜索互联网内容，获取网页文本。

## 适用场景
- 用户要求搜索某个主题
- 用户要求获取某个网页的内容
- 需要获取最新信息

## 执行步骤
1. 确认搜索关键词或 URL
2. 执行搜索或获取网页
3. 返回结果

## 成功标准
- 返回搜索结果（标题、链接、摘要）
- 或返回网页文本内容

## 注意事项
- 需要设置 BOCHA_API_KEY 环境变量
- 网络不可用时会失败

## 相关工具
- `search`：搜索网页，返回标题、链接和摘要
- `fetch_page`：获取指定网页的文本内容
