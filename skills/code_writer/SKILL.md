# Code Writer Skill

## 目的
创建和修改代码文件，支持写入、读取、替换操作。

## 适用场景
- 用户要求写一段代码并保存
- 用户要求读取某个文件的内容
- 用户要求修改文件中的某段代码

## 执行步骤
1. 确认文件路径
2. 执行写入/读取/修改操作
3. 返回操作结果

## 成功标准
- 文件已写入/读取/修改
- 返回文件路径和行数

## 相关工具
- `write_code`：将代码写入指定文件，自动创建目录
- `read_code`：读取指定文件的代码内容
- `modify_code`：修改指定文件中的代码，替换 old_text 为 new_text
