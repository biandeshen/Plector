# Memory Skill

## 目的
管理 Agent 的记忆系统，包括对话历史、用户偏好和知识记忆。

## 适用场景
- 用户要求记住某些信息
- 用户询问之前聊过的内容
- 用户设置偏好（如语言、格式）
- Agent 需要查询历史知识

## 执行步骤
1. 确认记忆类型（对话/偏好/知识）
2. 执行存储或查询操作
3. 返回结果

## 成功标准
- 对话历史已保存/已查询
- 用户偏好已保存/已获取
- 知识记忆已保存/已搜索

## 相关工具
- `save_conversation`：保存对话记录
- `get_conversation_history`：获取指定会话的对话历史
- `save_preference`：保存用户偏好
- `get_preference`：获取用户偏好
- `save_knowledge`：保存知识记忆
- `search_knowledge`：搜索知识记忆

## 注意事项
- 对话历史按 session_id 隔离，不会跨会话泄漏
- 偏好设置最多加载 20 条，避免上下文膨胀
- 对话内容预览最多 100 字符
- 知识搜索使用 LIKE 模糊匹配
