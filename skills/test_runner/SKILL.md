# Test Runner Skill

## 目的
运行 pytest 测试或任意 shell 命令，返回执行结果。

## 适用场景
- 用户要求运行单元测试
- 用户要求执行某个命令
- 代码修改后验证测试是否通过

## 执行步骤
1. 确认测试路径或命令
2. 执行测试/命令
3. 返回执行结果

## 成功标准
- 测试已运行
- 返回通过/失败数量
- 返回命令输出

## 相关工具
- `run_tests`：运行 pytest 测试，返回通过/失败结果
- `run_command`：运行任意 shell 命令并返回输出
