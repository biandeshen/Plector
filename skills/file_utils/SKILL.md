# File Utils Skill

## 目的
安全地操作文件系统，支持列表、复制、移动、删除、读取操作。

## 适用场景
- 用户要求列出目录内容
- 用户要求复制/移动/删除文件
- 用户要求读取文件内容

## 执行步骤
1. 确认操作类型和路径
2. 检查路径安全性
3. 执行操作
4. 返回结果

## 成功标准
- 操作成功完成
- 返回操作结果

## 注意事项
- 禁止操作受保护路径（/、C:\、/etc 等）
- 路径检查通过后才执行操作

## 相关工具
- `list_files`：列出目录下的文件和子目录
- `copy_file`：复制文件到目标路径
- `move_file`：移动文件到目标路径
- `delete_file`：删除指定文件
- `read_file`：读取文件内容
