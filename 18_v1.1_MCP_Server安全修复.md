## 第一步：修复安全漏洞

修改 `servers/filesystem_server.py` 中的 `check_safe_path` 函数：

找到：

```python
def check_safe_path(filepath: str) -> Path:
    """检查路径是否安全"""
    path = (ROOT_DIR / filepath).resolve()
    # 确保路径在根目录内
    if not str(path).startswith(str(ROOT_DIR)):
        raise PermissionError(f"路径超出根目录范围: {filepath}")
    # 确保不操作受保护路径
    resolved = str(path)
    for forbidden in FORBIDDEN_PATHS:
        if resolved == forbidden or resolved.startswith(forbidden + os.sep):
            raise PermissionError(f"禁止操作受保护路径: {resolved}")
    return path
```

替换为：

```python
def check_safe_path(filepath: str) -> Path:
    """
    检查路径是否安全

    安全规则：
        1. 路径必须在 ROOT_DIR 内（使用 relative_to，比 startswith 更安全）
        2. 路径不能指向受保护的系统路径

    参数:
        filepath: 相对于 ROOT_DIR 的文件路径

    返回:
        解析后的绝对路径

    异常:
        PermissionError: 路径不安全
    """
    path = (ROOT_DIR / filepath).resolve()

    # 使用 relative_to 检查（比 startswith 更安全，兼容 Windows 大小写）
    try:
        path.relative_to(ROOT_DIR)
    except ValueError:
        raise PermissionError(f"路径超出根目录范围: {filepath}")

    # 确保不操作受保护路径
    resolved = str(path)
    for forbidden in FORBIDDEN_PATHS:
        if resolved == forbidden or resolved.startswith(forbidden + os.sep):
            raise PermissionError(f"禁止操作受保护路径: {resolved}")

    return path
```

## 第二步：落实审查建议项

### 2.1 命令行参数验证

修改 `servers/filesystem_server.py` 顶部：

找到：

```python
ROOT_DIR = Path(sys.argv[1]) if len(sys.argv) > 1 else Path(".")
ROOT_DIR = ROOT_DIR.resolve()
```

替换为：

```python
# 从命令行参数获取根目录，默认为当前目录
if len(sys.argv) > 1:
    ROOT_DIR = Path(sys.argv[1]).resolve()
    if not ROOT_DIR.exists():
        sys.exit(f"错误: 根目录不存在: {ROOT_DIR}")
    if not ROOT_DIR.is_dir():
        sys.exit(f"错误: 不是目录: {ROOT_DIR}")
else:
    ROOT_DIR = Path(".").resolve()
```

### 2.2 搜索结果限制说明

找到：

```python
    for m in matches[:50]:
```

替换为：

```python
    # 限制返回 50 个结果，避免输出过大影响 LLM 上下文
    for m in matches[:50]:
```

### 2.3 错误处理细化

找到 `call_tool` 方法中的异常处理：

```python
    except Exception as e:
        return [TextContent(type="text", text=f"错误: {e}")]
```

替换为：

```python
    except PermissionError as e:
        return [TextContent(type="text", text=f"权限错误: {e}")]
    except FileNotFoundError as e:
        return [TextContent(type="text", text=f"文件不存在: {e}")]
    except Exception as e:
        return [TextContent(type="text", text=f"错误: {type(e).__name__}: {e}")]
```

**文件**：`core/mcp_client.py`

**职责**：通过 MCP 协议连接外部工具服务器，发现并调用远程工具。

**关键设计**：
- MCPServer：管理单个 MCP Server 连接（stdio 传输）

- 懒加载：首次调用 AgentLoop.run() 时才初始化 MCP 连接
