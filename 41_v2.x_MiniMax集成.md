## 第二步：修改 MCP 客户端支持环境变量

修改 `core/mcp_client.py` 中的 `connect` 方法：

```python
    async def connect(self, config: dict):
        """连接 MCP Server"""
        command = config["command"]
        args = config.get("args", [])
        env = config.get("env", {})

        # 合并环境变量
        import os
        full_env = {**os.environ, **env}

        self.process = await asyncio.create_subprocess_exec(
            command, *args,
            stdin=asyncio.subprocess.PIPE,
            stdout=asyncio.subprocess.PIPE,
            stderr=asyncio.subprocess.PIPE,
            env=full_env,
        )
```

## 第三步：修改 MCP 管理器支持环境变量

修改 `core/mcp_manager.py` 中的 `load_config` 方法，确保环境变量正确传递：

```python
    async def load_config(self, config_path: str = "config.yaml"):
        """加载 MCP Server 配置"""
        import yaml

        with open(config_path, "r") as f:
            config = yaml.safe_load(f)

        mcp_config = config.get("mcp_servers", {})

        for name, server_config in mcp_config.items():
            client = MCPClient(name)
            await client.connect(server_config)
            self.servers[name] = client

            # 注册工具
            tools = await client.list_tools()
            for tool in tools:
                self.tool_registry[name] = tool

            logger.info(f"MCP Server {name} 已连接，工具数: {len(tools)}")
```

## 第五步：更新 MCP 工具注册逻辑

修改 `core/mcp_manager.py`，确保 MiniMax 工具正确注册到技能系统：

```python
    def get_all_tools(self) -> list:
        """获取所有 MCP 工具（用于注册到技能系统）"""
        tools = []
        for server_name, tool in self.tool_registry.items():
            tools.append({
                "name": tool["name"],
                "description": tool.get("description", ""),
                "inputSchema": tool.get("inputSchema", {}),
                "server": server_name,
            })
        return tools
```

在 `core/skill_registry.py` 中，确保 MCP 工具加载时机正确：

```python
    async def load_mcp_tools(self):
        """加载 MCP 工具到技能注册表"""
        from core.mcp_manager import MCPManager

        manager = MCPManager()
        await manager.load_config()

        tools = manager.get_all_tools()
        for tool in tools:
            self.register_mcp_tool(
                server=tool["server"],
                name=tool["name"],
                description=tool["description"],
                input_schema=tool["inputSchema"],
            )

        logger.info(f"MCP 工具加载完成: {len(tools)} 个")
```

## MiniMax 工具说明

| 工具 | 功能 | 参数 |
|------|------|------|
| web_search | 网络搜索 | query（搜索词） |
| understand_image | 图片理解 | prompt（提问）+ image_url（图片 URL 或本地路径） |

支持图片格式：JPEG、PNG、GIF、WebP（最大 20MB）

## 图片理解使用示例

```
用户：分析一下这张架构图 https://example.com/arch.png

  1. 调用 minimax.understand_image

  3. image_url: "https://example.com/arch.png"
  4. 返回分析结果
