# Plector OpenAI strict 模式兼容修复（Claude Code 可执行版）

> 共 2 步，按顺序执行。

## 第二步：修复所有 implementation.py（处理 null 值）

### 2.1 skills/file_utils/implementation.py

找到 `list_files` 方法：

```python
    async def list_files(self, path: str = ".", pattern: str = "*") -> Dict[str, Any]:
```

替换为：

```python
    async def list_files(self, path=None, pattern=None) -> Dict[str, Any]:
        """列出目录下的文件"""
        if path is None:
            path = "."
        if pattern is None:
            pattern = "*"
```

找到 `read_file` 方法：

```python
    async def read_file(self, filepath: str, max_lines: int = 100) -> Dict[str, Any]:
```

替换为：

```python
    async def read_file(self, filepath: str, max_lines=None) -> Dict[str, Any]:
        """读取文件内容"""
        if max_lines is None:
            max_lines = 100
```

### 2.2 skills/test_runner/implementation.py

找到 `run_tests` 方法：

```python
    async def run_tests(self, path: str = "tests/") -> Dict[str, Any]:
```

替换为：

```python
    async def run_tests(self, path=None) -> Dict[str, Any]:
        """运行 pytest 测试"""
        if path is None:
            path = "tests/"
```

找到 `run_command` 方法：

```python
    async def run_command(self, command: str, timeout: int = 30) -> Dict[str, Any]:
```

替换为：

```python
    async def run_command(self, command: str, timeout=None) -> Dict[str, Any]:
        """运行任意 shell 命令"""
        if timeout is None:
            timeout = 30
```

### 2.3 skills/web_search/implementation.py

找到 `search` 方法：

```python
    async def search(self, query: str, count: int = 5) -> Dict[str, Any]:
```

替换为：

```python
    async def search(self, query: str, count=None) -> Dict[str, Any]:
        """搜索网页"""
        if count is None:
            count = 5
```

## 根本原因

```
OpenAI strict 模式规则：
  required 必须包含 properties 中的所有 key
  不能有可选参数

Plector 之前的做法：
  有默认值的参数不在 required 中
  → OpenAI 拒绝

修复后：
  所有参数都在 required 中
  LLM 可能传 null
  implementation.py 处理 null → 使用默认值
```
